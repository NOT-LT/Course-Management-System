Essential Tailwind Classes

Spacing

  • p-4 = padding: 1rem
  • px-6 = padding left/right: 1.5rem
  • py-3 = padding top/bottom: 0.75rem
  • m-4 = margin: 1rem
  • mb-6 = margin-bottom: 1.5rem
  • gap-4 = gap: 1rem (for flex/grid)

Layout

  • flex = display: flex
  • grid = display: grid
  • grid-cols-2 = 2 columns
  • items-center = align-items: center
  • justify-between = justify-content: space-between
  • max-w-7xl = max-width: 80rem

Typography

  • text-xl = font-size: 1.25rem
  • font-bold = font-weight: 700
  • text-center = text-align: center
  • text-foreground = our text color
  • text-muted-foreground = subtle text

Borders & Rounding

  • border = border: 1px solid
  • border-2 = border: 2px solid
  • border-border = our border color
  • rounded-lg = border-radius: 0.5rem
  • rounded-full = fully rounded

Responsive Design

Use breakpoint prefixes to create responsive layouts:

md:grid-cols-2

2 columns on medium screens and up

lg:px-8

Larger padding on large screens

hidden md:block

Hide on mobile, show on medium+ screens

<div class="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-4">
  <!-- 1 column on mobile, 2 on tablet, 3 on desktop -->
</div>
